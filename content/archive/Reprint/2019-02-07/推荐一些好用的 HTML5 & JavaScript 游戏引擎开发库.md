<h1 id="articleHeader0">推荐一些好用的 HTML5 &amp; JavaScript 游戏引擎开发库</h1>
<h2 id="articleHeader1">0. 引言</h2>
<ul>
<li><p>如果你是一个游戏开发者，并且正在寻找一个可以与 JavaScript 和 HTML5 无缝工作的游戏引擎。那么在这里你会找到你需要的东西。尽管市面上有很多免费和付费的游戏引擎，但一个让专业的游戏开发者满意的游戏引擎肯定需要包含独特的东西。游戏开发者对完美游戏引擎的追求永远不会停止。</p></li>
<li><p>下面将介绍最好的html5和javascript游戏引擎列表，每个游戏引擎有每个人想要的特性。每个游戏引擎有他自己的优缺点，这样你可以根据你自己的需求做选择，我确定这个列表中有你在寻找的游戏引擎</p></li>
</ul>
<h2 id="articleHeader2">1. Canvas Engine</h2>
<ul><li><p>Canvas Engine 提供创建html5视频游戏的平台，创建html5游戏，它会运行在所有的现代浏览器，平板电脑和智能手机上。<br><span class="img-wrap"><img data-src="/img/remote/1460000006781199" src="https://static.alili.tech/img/remote/1460000006781199" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
<h2 id="articleHeader3">2. HTML5 Quintus</h2>
<ul><li><p>Quintus 是另一个容易使用，适用于移动设备和桌面的javascript html5游戏引擎。 Quintus有一个模块引擎，通过它你可以仅仅引用你需要的模块，并在，在一个页面上，可以运行多个实例。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989873" src="https://static.alili.tech/img/remote/1460000005989873" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
<h2 id="articleHeader4">3. Turbulenz</h2>
<ul><li><p>Trubulenz 是一个开源的用来创建使用高级技术和web服务的HTML5游戏引擎。引擎库使用支持游戏代码数据快速迭代的优化后的javascritp实现。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989875" src="https://static.alili.tech/img/remote/1460000005989875" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
<h2 id="articleHeader5">4. Squarepig</h2>
<ul><li><p>这是一个非常容易使用的游戏引擎，它适合新手去创建他们第一个web 游戏，也适合有经验的程序猿创建示例和原型。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989879" src="https://static.alili.tech/img/remote/1460000005989879" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
<h2 id="articleHeader6">5. Akihabara</h2>
<ul><li><p>Akihabara 是一个html5游戏引擎，同样也是一组用javascript创建运行在浏览器中的不依赖falsh技术的马赛克风格的8/16位游戏的库和工具。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989881" src="https://static.alili.tech/img/remote/1460000005989881" alt="" title="" style="cursor: pointer;"></span></p></li></ul>
<h2 id="articleHeader7">6. GoGoMakePlay</h2>
<ul><li><p>GMP 是免费的、快速的Javascript游戏引擎，它轻巧而简便易懂。 如果你想要做一些基于精灵的2-D游戏，GMP是很棒的选择。它可以很容易的进行大多数复古风格的游戏设计。它也可以像sudoku一样制作拼图游戏。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989883" src="https://static.alili.tech/img/remote/1460000005989883" alt="" title="" style="cursor: pointer;"></span></p></li></ul>
<h2 id="articleHeader8">7. Traffic Cone</h2>
<ul><li><p>Traffic Cone支持多人游戏，目前只提供客户端支持，所以你需要提供自己的服务器。为了在不久之后提供更多支持，我们正积极致力于Traffic Cone服务端的开发工作。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989885" src="https://static.alili.tech/img/remote/1460000005989885" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
<h2 id="articleHeader9">8. Collie</h2>
<ul>
<li><p>Collie是一个Javascript库，它有助于让HTML5创建高度优化的动画和游戏。 Collie可以在PC 和移动平台上运行HTML 5 canvas 和 DOM。</p></li>
<li><p>Collie可以稳定地使用渲染管线处理多个对象，并且支持一些有益的特性包括动画精灵和用户事件。它稳定支持iOS和Android，并且为每一个平台优化渲染。 视网膜屏幕也可以轻易地响应。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989887" src="https://static.alili.tech/img/remote/1460000005989887" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li>
</ul>
<h2 id="articleHeader10">9. Gamejs</h2>
<ul><li><p>GameJs是一个基于HTML canvas元素的轻量级库。特别值得一提的是，它的绘画函数为游戏开发者提供各种各样的有用模块，并且还在不断扩充之中。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989889" src="https://static.alili.tech/img/remote/1460000005989889" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>

<h2 id="articleHeader15">14. Play Craftlabs</h2>
<ul><li><p>Playcraft具备了一个完整的HTML5游戏引擎工作集，提供了你在建立游戏并直接发布到市场上的每一件工具。这是一个独一无二的引擎，当编写游戏时你可以轻松地将其转换到许多不同平台上，包括Facebook，简朴的老式网站，以及一些本机化的Android和iOS应用。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989899" src="https://static.alili.tech/img/remote/1460000005989899" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
<h2 id="articleHeader16">15. Gladiusjs</h2>
<ul><li><p>Gladius是一个3D游戏引擎，全部用JavaScript编写，设计于在浏览器上跑的。该引擎包括了一个功能核心集，这对于所有游戏是通用的，并且还模拟了游戏回环、消息、任务、定时器等。像空间变换这样的通用组件也在内核中提供了。而绘图和物理学这样的特殊功能也作为游戏扩展封装起来并设计于运行在内核之上。一个通用的扩展集将作为工程的一部分被维护着，并且该引擎的一个强大设计目标就是支持第三方扩展。<br><span class="img-wrap"><img data-src="/img/remote/1460000005989901" src="https://static.alili.tech/img/remote/1460000005989901" alt="" title="" style="cursor: pointer; display: inline;"></span></p></li></ul>
