<p>2.CSShake<br>说明：用CSS实现各种颠颤动画效果DOM！<br>GitHub主页： <a href="https://github.com/elrumordelaluz/csshake" rel="nofollow noreferrer" target="_blank">https://github.com/elrumordel...</a></p>
<p>3.Hover.css<br>说明：将鼠标Hover效果快速应用在你的网页元素或创意设计上。<br>GitHub主页：<a href="https://github.com/IanLunn/Hover" rel="nofollow noreferrer" target="_blank">https://github.com/IanLunn/Hover</a></p>
<p>4.Velocity.js<br>说明：帮你加速JavaScript动画。<br>GitHub主页：<a href="https://github.com/julianshapiro/velocity" rel="nofollow noreferrer" target="_blank">https://github.com/julianshap...</a></p>
<p>5.DynCSS<br>说明：用动态CSS来打造你的网页。<br>GitHub主页：<a href="https://github.com/vzaccaria/DynCSS" rel="nofollow noreferrer" target="_blank">https://github.com/vzaccaria/...</a></p>
<p>6.Magic Animations<br>说明：可创造CSS3动画特效。<br>GitHub主页：<a href="https://github.com/miniMAC/magic" rel="nofollow noreferrer" target="_blank">https://github.com/miniMAC/magic</a></p>
<p>7.Bounce.js<br>说明：即时创造漂亮的CSS3动画。<br>GitHub主页： <a href="https://github.com/tictail/bounce.js" rel="nofollow noreferrer" target="_blank">https://github.com/tictail/bo...</a></p>
<p><strong>前端框架和类库</strong><br>1.lodash.js<br>简介：具有一致接口、模块化、高性能等特性的 JavaScript 工具库<br>GitHub主页：<a href="https://github.com/lodash/lodash/" rel="nofollow noreferrer" target="_blank">https://github.com/lodash/lod...</a></p>
<p>2.Underscore.js<br>简介：JavaScript 工具库，它提供了一整套函数式编程的实用功能，但是没有扩展任何 JavaScript 内置对象。 他解决了这个问题：“如果我面对一个空白的 HTML 页面，并希望立即开始工作，我需要什么？” 他弥补了 jQuery 没有实现的功能，同时又是 Backbone 必不可少的部分。</p>
<p>3.AngularJS<br>简介：AngularJS是为了克服HTML在构建应用上的不足而设计的。HTML是一门很好的为静态文本展示设计的声明式语言，但要构建WEB应用的话它就显得乏力了。<br>GitHub主页：<a href="https://github.com/angular/angular.js" rel="nofollow noreferrer" target="_blank">https://github.com/angular/an...</a></p>
<p>4.React<br>简介：React 是一个用于构建用户界面的 JAVASCRIPT 库。主要用于构建UI，很多人认为 React 是 MVC 中的 V（视图）。<br>GitHub主页：<a href="https://github.com/facebook/react" rel="nofollow noreferrer" target="_blank">https://github.com/facebook/r...</a></p>
<p>5.Vue.js<br>简介：Vue.js是一个用于构建用户界面的轻量级渐进框架。它提供了一个类似React的虚拟DOM驱动视图层，可以与其它类库集成，用于构建强大的单页面应用。该框架是由之前在AngularJS工作过的Evan You创建的，他提取了AngularJS中自己喜欢的部分。</p>
<p>Vue.js使用HTML模板语法将DOM绑定到实例数据。Model是在数据改变时更新view的纯JavaScript对象。<br>GitHub主页：<a href="https://github.com/vuejs/vue" rel="nofollow noreferrer" target="_blank">https://github.com/vuejs/vue</a><br>官方地址：<a href="https://cn.vuejs.org/" rel="nofollow noreferrer" target="_blank">https://cn.vuejs.org/</a></p>
<p>6.Backbone.js<br>简介：Backbone.js是提供服务器端框架中常见的MVC结构的最早客户端选项之一。它唯一的依赖是Underscore.js。<br>Backbone.js声称是一个类库，因为它可以与其它项目集成，但是我觉得大多数开发人员都认为它是一个框架。<br>GitHub主页：<a href="https://github.com/jashkenas/backbone/" rel="nofollow noreferrer" target="_blank">https://github.com/jashkenas/...</a></p>
<p>7.Ember.js<br>简介：Ember.js是基于Model-View-ViewModel（MVVM）模式的框架之一。它在单个包中实现模板化、数据绑定和类库。<br>GitHub主页：<a href="https://github.com/emberjs/ember.js" rel="nofollow noreferrer" target="_blank">https://github.com/emberjs/em...</a></p>
<p><strong>前端工具</strong><br>1.Gulp.js<br>简介：虽然Gulp不是第一个任务执行工具，它却很快成为了最受欢迎的。Gulp使用易于阅读的JavaScript代码，将源文件加载到流中，并在将数据输出到构建文件夹之前，通过各种插件管理数据。在任何其它选项之前检查Gulp.js是简单、快速和有趣的。<br>GitHub主页：<a href="https://github.com/gulpjs/gulp" rel="nofollow noreferrer" target="_blank">https://github.com/gulpjs/gulp</a></p>
<p>2.NPM<br>简介：npm是Node.js包管理器，但其脚本工具可用于运行通用任务。对于具有少量依赖关系的简单项目来说，这是一个很好的选择。然而，更复杂的任务可能会变得不太适用。<br>GitHub主页：<a href="https://github.com/npm/npm" rel="nofollow noreferrer" target="_blank">https://github.com/npm/npm</a></p>
<p>3.Grunt<br>简介：Grunt是第一批被大规模采用的JavaScript任务执行工具之一，但其速度和复杂的JSON配置却导致了Gulp的兴起。现在这些问题已经得到了解决，Grunt仍然是一个受欢迎的选择。<br>GitHub主页：<a href="https://github.com/gruntjs/grunt" rel="nofollow noreferrer" target="_blank">https://github.com/gruntjs/grunt</a></p>
<p>4.WebPack<br>简介：Webpack支持所有流行的模块选项，并已成为React开发的代名词。虽然Webpack声称是一个模块捆绑程序，但是已经可以用作通用任务运行程序了。<br>GitHub主页：<a href="https://github.com/webpack/webpack" rel="nofollow noreferrer" target="_blank">https://github.com/webpack/we...</a></p>
<p>5.Browserify<br>简介：Browserify支持Node.js正在使用的CommonJS模块，它将所有模块编译成单个浏览器兼容的文件。<br>GitHub主页：<a href="https://github.com/substack/node-browserify" rel="nofollow noreferrer" target="_blank">https://github.com/substack/n...</a></p>
<p>6.RequireJS<br>简介：RequireJS是一种浏览器中的模块加载器，它也可以在Node.js中使用。<br>GitHub主页：<a href="https://github.com/jrburke/r.js" rel="nofollow noreferrer" target="_blank">https://github.com/jrburke/r.js</a></p>
<p>7.ESLint<br>简介：ESLint是一种可插拔的代码分析工具。每个规则都是一个插件，因此可以根据个人喜好进行配置。<br>GitHub主页：<a href="https://github.com/eslint/eslint" rel="nofollow noreferrer" target="_blank">https://github.com/eslint/eslint</a></p>
<p>8.JSHint<br>简介：JSHint是一个灵活的JavaScript代码分析工具，它很好的平衡了真正的错误和老旧的语法。<br>GitHub主页：<a href="https://github.com/jshint/jshint" rel="nofollow noreferrer" target="_blank">https://github.com/jshint/jshint</a></p>
<p>9.JSLint<br>简介：JSLint是最早的代码分析工具之一，它实现了一套严格的默认规则。<br>GitHub主页：<a href="https://github.com/reid/node-jslint" rel="nofollow noreferrer" target="_blank">https://github.com/reid/node-...</a></p>
<p>10.Jasmine<br>简介：Jasmine是一个行为驱动的测试工具，可以在浏览器中自动测试UI和交互。<br>GitHub主页：<a href="https://github.com/jasmine/jasmine-npm" rel="nofollow noreferrer" target="_blank">https://github.com/jasmine/ja...</a></p>
<p>11.QUnit<br>简介：QUnit是一个单元测试框架，可以在输入特定参数时，检查函数结果。它还上报测试结果，确保没有错过特定的代码分支。<br>GitHub主页：<a href="https://github.com/kof/node-qunit" rel="nofollow noreferrer" target="_blank">https://github.com/kof/node-q...</a></p>
<p><strong>CSS框架</strong><br>1.Bootstrap – 最流行的Web前端UI框架<br>Bootstrap是由twitter推出的Web前端UI框架，它由Twitter的设计师Mark Otto和Jacob Thornton合作开发,是一个CSS/HTML框架。它使用了最新的浏览器技术，Bootstrap提供了时尚的排版样式，表单，buttons，表格，网格系统等等。<br>官方网站：<a href="http://getbootstrap.com/" rel="nofollow noreferrer" target="_blank">http://getbootstrap.com/</a></p>
<p>2.jQuery UI Bootstrap<br>它是jQuery UI和Bootstrap的集成，它是Bootstrap样式的，因此外观比较漂亮，同时它拥有jQuery UI的控件功能，这也方便开发者快速地创建一个网页控件。</p>
<p>3.BootMetro - Metro风格的CSS框架<br>BootMetro是一款基于Bootstrap的前端UI框架，BootMetro的特点是可以很方便地构建类似Windonws 8扁平化风格的网页界面，效果非常不错。<br>官方网站：<a href="http://aozora.github.io/bootmetro/" rel="nofollow noreferrer" target="_blank">http://aozora.github.io/bootm...</a></p>
<p>4.Flat UI - 扁平风格 UI 工具包<br>Flat UI是一套精美的扁平风格 UI 工具包，基于 Twitter Bootstrap 实现。这套界面工具包含许多基本的和复杂的 UI 部件，例如按钮，输入框，组合按钮，复选框，单选按钮，标签，菜单，进度条和滑块，导航元素等等。<br>官方网站：<a href="https://github.com/designmodo/Flat-UI" rel="nofollow noreferrer" target="_blank">https://github.com/designmodo...</a></p>
<p>5.网易CSS框架 NEC<br>NEC是网易推出的开源前端CSS框架，NEC提供了丰富UI代码库和插件，可以极大的帮助开发人员提高开发效率。即使你并非前端专业开发人员，利用NEC你也可以快速地构建属于自己的网页应用。<br>官方网站：<a href="http://nec.netease.com/" rel="nofollow noreferrer" target="_blank">http://nec.netease.com/</a></p>
<p>6.Alloy UI – 功能强大的CSS UI框架<br>Alloy UI是基于YUI 3的前段UI框架，包含一套丰富的（超过60）UI 部件，如图片库，对话框，树形结构，面板，自动完成，按钮，日历控件，工具条等。<br>官方网站：<a href="http://alloyui.com/" rel="nofollow noreferrer" target="_blank">http://alloyui.com/</a></p>
<p>7.Cardinal – 移动端的CSS UI框架<br>Cardinal 是一个小型的移动优先的 CSS 框架，提供很多有用的默认样式、可缩放排版、可重用模块和一个简单的响应式表格系统。<br>官方网站：<a href="http://cardinalcss.com/" rel="nofollow noreferrer" target="_blank">http://cardinalcss.com/</a></p>
<p>8.快速开发CSS的框架 CSScaffold<br>不同于许多CSS框架，它必须依靠PHP与Apache的mod_rewrite来执行，但也因为需要这两种东西，让CSScaffold变得很神奇、很方便，写起CSS来又快又轻松！<br>官方网站：<a href="http://www.w3avenue.com/2009/10/13/csscaffold-php-based-css-framework/" rel="nofollow noreferrer" target="_blank">http://www.w3avenue.com/2009/...</a></p>
<p>9.后台UI开发框架 MuseUI<br>一款基于bootstrap风格，兼容于主流浏览器（包括IE6）的后端UI开发组件。<br>官方网站：<a href="http://git.oschina.net/muse/museui" rel="nofollow noreferrer" target="_blank">http://git.oschina.net/muse/m...</a></p>
