<h4>项目特点</h4>
<ul>
<li>[x] 基于 GitHub Pages 无需服务器</li>
<li>[x] 改进传统 GitHub Pages 不能动态发布的缺陷</li>
<li>[x] 无需任何软件 快速搭建</li>
<li>[x] 包含电脑端和移动端</li>
<li>[x] 单页面应用</li>
</ul>
