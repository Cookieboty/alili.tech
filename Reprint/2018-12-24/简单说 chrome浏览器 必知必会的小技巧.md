<h3 id="articleHeader2">区域截屏</h3>
<p>选取页面中的一部分，保存为图片   <br>1、打开开发者工具   <br>使用 快捷键 F12 (Windows) 或 Cmd+Opt+I (Mac)  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7bh?w=787&amp;h=429" src="https://static.alili.tech/img/bVY7bh?w=787&amp;h=429" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>2、选择左上角的元素选择按钮，图标颜色变为蓝色即表示选中了  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7bg?w=269&amp;h=165" src="https://static.alili.tech/img/bVY7bg?w=269&amp;h=165" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>3、Windows 下按住 Ctrl，Mac 就按住 Command，然后点击鼠标左键在页面选择区域即可，松开鼠标后，截图自动下载。   </p>
<p>4、图片自动下载好后，点击图片后面的箭头，可以在文件夹中显示  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7aq?w=417&amp;h=328" src="https://static.alili.tech/img/bVY7aq?w=417&amp;h=328" alt="图片描述" title="图片描述" style="cursor: pointer;"></span></p>
<h3 id="articleHeader3">节点截图</h3>
<p>选中页面中某一元素，保存为图片   <br>1、打开开发者工具   <br>使用 快捷键 F12 (Windows) 或 Cmd+Opt+I (Mac)</p>
<p><span class="img-wrap"><img data-src="/img/bVY7bh?w=787&amp;h=429" src="https://static.alili.tech/img/bVY7bh?w=787&amp;h=429" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>2、选中任意元素节点  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7at?w=551&amp;h=175" src="https://static.alili.tech/img/bVY7at?w=551&amp;h=175" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>3、打开命令工具<br>使用快捷键 Ctrl + Shift + p (Windows) 或 Cmd + Shift + p (Mac)  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7au?w=568&amp;h=327" src="https://static.alili.tech/img/bVY7au?w=568&amp;h=327" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>4、点击Capture node screenshot，或者输入这行中任意的关键字，比如输入node，也会出来这个选项，然后点击这个选项，图片会自动下载。 </p>
<p><span class="img-wrap"><img data-src="/img/bVY7av?w=531&amp;h=159" src="https://static.alili.tech/img/bVY7av?w=531&amp;h=159" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>

<h3 id="articleHeader4">截全屏</h3>
<p>保存完整网页为图片   <br><strong>第一种方式</strong><br>1、打开开发者工具   <br>使用 快捷键 F12 (Windows) 或 Cmd+Opt+I (Mac)</p>
<p><span class="img-wrap"><img data-src="/img/bVY7bh?w=787&amp;h=429" src="https://static.alili.tech/img/bVY7bh?w=787&amp;h=429" alt="图片描述" title="图片描述" style="cursor: pointer;"></span></p>
<p>2、打开命令工具<br>使用快捷键 Ctrl + Shift + p (Windows) 或 Cmd + Shift + p (Mac)  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7au?w=568&amp;h=327" src="https://static.alili.tech/img/bVY7au?w=568&amp;h=327" alt="图片描述" title="图片描述" style="cursor: pointer;"></span></p>
<p>3、点击Capture full size screenshot，或者输入这行中任意的关键字，比如输入full，也会出来这个选项     </p>
<p><span class="img-wrap"><img data-src="/img/bVY7as?w=523&amp;h=110" src="https://static.alili.tech/img/bVY7as?w=523&amp;h=110" alt="图片描述" title="图片描述" style="cursor: pointer;"></span></p>
<p>4、图片自动下载好后，点击图片后面的箭头，可以在文件夹中显示  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7aq?w=417&amp;h=328" src="https://static.alili.tech/img/bVY7aq?w=417&amp;h=328" alt="图片描述" title="图片描述" style="cursor: pointer;"></span></p>
<p><strong>第二种方式</strong><br>1、打开开发者工具   <br>使用 快捷键 F12 (Windows) 或 Cmd+Opt+I (Mac)  </p>
<p>2、选中  切换开发模式按钮，图标颜色变为蓝色即表示选中了  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7ap?w=235&amp;h=140" src="https://static.alili.tech/img/bVY7ap?w=235&amp;h=140" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>

<h3 id="articleHeader5">拾色器</h3>
<p>在Styles边栏 点击任意颜色的小色块，就可以弹出颜色选择器</p>
<p><span class="img-wrap"><img data-src="/img/bVY7ao?w=280&amp;h=261" src="https://static.alili.tech/img/bVY7ao?w=280&amp;h=261" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p><span class="img-wrap"><img data-src="/img/bVY7bf?w=815&amp;h=556" src="https://static.alili.tech/img/bVY7bf?w=815&amp;h=556" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span><br>1、颜色选择区域。<br>2、吸管。<br>3、复制到剪贴板。将显示值复制到剪贴板。<br>4、显示值。颜色的RGBA，HSLA或十六进制表示。<br>5、调色板。单击其中一个方块将颜色更改为该方块。<br>6、色相。<br>7、透明度。<br>8、显示值切换器。在当前颜色的RGBA，HSLA和Hex表示之间切换。<br>9、调色板切换器。</p>
<h3 id="articleHeader6">快速添加样式规则</h3>
<p>1、在Styles 边栏，鼠标放在每一小块样式规则上，右下方都会有三个小点   </p>
<p><span class="img-wrap"><img data-src="/img/bVY7an?w=471&amp;h=295" src="https://static.alili.tech/img/bVY7an?w=471&amp;h=295" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>2、鼠标放在这三个小点上，会出现5个小图标，每个小图标都有作用  </p>
<p><span class="img-wrap"><img data-src="/img/bVY7am?w=444&amp;h=139" src="https://static.alili.tech/img/bVY7am?w=444&amp;h=139" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
<p>3、他们从左到右分别代表</p>
<ul>
<li>添加 text-shadow</li>
<li>添加 box-shadow</li>
<li>添加 color</li>
<li>添加 background-color</li>
<li>插入样式规则</li>
</ul>
<h3 id="articleHeader7">增加移动设备</h3>
<p>1、打开开发者工具   <br>使用 快捷键 F12 (Windows) 或 Cmd+Opt+I (Mac)</p>
<p>2、点击右上方的三个小点，然后选择Settings </p>
<p><span class="img-wrap"><img data-src="/img/bVY7al?w=812&amp;h=261" src="https://static.alili.tech/img/bVY7al?w=812&amp;h=261" alt="图片描述" title="图片描述" style="cursor: pointer;"></span></p>
<p>3、选择Devices，然后在需要添加的设备前面打上勾就可以了   </p>
<p><span class="img-wrap"><img data-src="/img/bVY7ak?w=505&amp;h=343" src="https://static.alili.tech/img/bVY7ak?w=505&amp;h=343" alt="图片描述" title="图片描述" style="cursor: pointer; display: inline;"></span></p>
