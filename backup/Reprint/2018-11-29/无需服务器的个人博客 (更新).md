<h2 id="articleHeader0">简介</h2>
<p>博客基于 GitHub Pages 与 Github API 实现无后台，可动态发布博客的系统<br>博客数据储存于gist 通过Github API 进行增删改查</p>

<h4>项目特点</h4>
<ul>
<li>[x] 基于 GitHub Pages 无需服务器</li>
<li>[x] 改进传统 GitHub Pages 不能动态发布的缺陷</li>
<li>[x] 无需任何软件 快速搭建</li>
<li>[x] 包含电脑端和移动端</li>
<li>[x] 单页面应用</li>
</ul>
