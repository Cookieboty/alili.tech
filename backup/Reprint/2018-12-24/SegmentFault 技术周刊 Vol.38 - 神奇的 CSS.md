<p>CSS 的全称是 Cascading Style Sheets，即层叠样式表， 是一种样式表语言，用来描述 HTML 或 XML（包括如 SVG、XHTML 之类的 XML 分支语言）元素所呈现的样式。</p>
<p>“层叠”即表示允许以多种方式来描述样式，一个 HTML 元素可以被渲染呈现出多种样式。</p>
<p>下面就让我们来看看 CSS 可以有多神奇吧。</p>
