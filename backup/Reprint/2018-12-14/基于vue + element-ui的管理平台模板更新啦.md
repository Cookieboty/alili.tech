<h1 id="articleHeader0">vue-element-admin-tpl</h1>
<p>这是基于vue + element-ui开发的管理平台模板。<br>基于该模板，你可以快速搭建起管理平台页面，基础架构已经搭建好。<br>当然，只要有时间，组件我会不断丰富。</p>
<p>本次更新：</p>
<ul>
<li>增加了主题</li>
<li>增加了响应式栅格布局</li>
<li>响应式导航</li>
</ul>
<h2 id="articleHeader1">已经安装依赖</h2>
<ul>
<li>vue</li>
<li>vuex</li>
<li>vue-router</li>
<li>axios</li>
<li>element-ui</li>
<li>nprogress</li>
<li>
<code>FontAwesome 4.7.0</code> 图标</li>
</ul>
